# 剪貼簿管理程式

一個輕量的 Windows 剪貼簿歷史記錄工具，使用 Python + Tkinter 製作。

## 功能

- 自動偵測剪貼板變化，即時記錄每一筆複製的內容
- 依日期分群顯示，可展開／收合
- 支援關鍵字搜尋
- 點擊「複製」可將舊記錄重新複製回剪貼板
- 點擊「刪除」可移除單筆記錄
- 歷史記錄存於本機 `clipboard_history.json`，重開程式後仍保留（最多 500 筆）

## 下載執行（免安裝）

至 [Releases](../../releases) 下載 `複製剪貼.exe`，直接點兩下執行即可，不需要安裝 Python。

> **Windows 安全警告說明**
> 首次執行時 Windows 可能會跳出「已保護您的電腦」的警告，這是因為此程式未購買商業數位簽署憑證所致，並非惡意程式。
> 請點「**更多資訊**」→「**仍要執行**」即可正常開啟。

## 環境需求（從原始碼執行）

- Python 3.8 以上
- 套件：`pyperclip`

```bash
pip install pyperclip
```

## 執行方式

```bash
python 複製剪貼.py
```
